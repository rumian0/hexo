# гҖҗи®Ўз®—жңәеҹәзЎҖгҖ‘е…ідәҺиҝӣеҲ¶

# иҝӣеҲ¶жҳҜд»Җд№Ҳ & еёёи§Ғзҡ„иҝӣеҲ¶

еӯҰи®Ўз®—жңәпјҢйӮЈе°ұиӮҜе®ҡеҫ—жҮӮдёҖж ·дёңиҘҝ вҖ”вҖ” иҝӣеҲ¶пјҲе“ӘжҖ•дҪ жІЎжңүеӯҰиҝҮпјҢеӨҡе°‘д№ҹдјҡжҮӮзӮ№пјү

йӮЈиҝӣеҲ¶жҳҜе•Ҙпјҹ

иҝӣеҲ¶д№ҹе°ұжҳҜиҝӣдҪҚи®Ўж•°еҲ¶пјҢжҳҜдәәдёәе®ҡд№үзҡ„еёҰиҝӣдҪҚзҡ„и®Ўж•°ж–№жі•пјҲжңүдёҚеёҰиҝӣдҪҚзҡ„и®Ўж•°ж–№жі•пјҢжҜ”еҰӮеҺҹе§Ӣзҡ„з»“з»іи®Ўж•°жі•пјҢе”ұзҘЁж—¶еёёз”Ёзҡ„ вҖңжӯЈвҖқ еӯ—и®Ўж•°жі•пјҢд»ҘеҸҠзұ»дјјзҡ„ tally mark и®Ўж•°пјүгҖӮ еҜ№дәҺд»»дҪ•дёҖз§ҚиҝӣеҲ¶ вҖ”X иҝӣеҲ¶пјҢе°ұиЎЁзӨәжҜҸдёҖдҪҚзҪ®дёҠзҡ„ж•°иҝҗз®—ж—¶йғҪжҳҜйҖў X иҝӣдёҖдҪҚгҖӮ еҚҒиҝӣеҲ¶жҳҜйҖўеҚҒиҝӣдёҖпјҢеҚҒе…ӯиҝӣеҲ¶жҳҜйҖўеҚҒе…ӯиҝӣдёҖпјҢдәҢиҝӣеҲ¶е°ұжҳҜйҖўдәҢиҝӣдёҖпјҢд»ҘжӯӨзұ»жҺЁпјҢx иҝӣеҲ¶е°ұжҳҜйҖў x иҝӣдҪҚгҖӮ

зҷҫеәҰзҷҫз§‘

иҝӣеҲ¶иҝҷзҺ©ж„ҸпјҢиҜҙзҷҪдәҶе°ұжҳҜжҲ‘д»¬ж•°ж•°зҡ„дёҖдёӘж–№ејҸпјҢеғҸе•ҘеҚҒиҝӣеҲ¶пјҲеӨ§е®¶йғҪзҹҘйҒ“зҡ„пјүе°ұжҳҜж»ЎеҚҒиҝӣдёҖ

йӮЈиҝӣеҲ¶жңүе•Ҙе‘ўпјҹеёёи§Ғзҡ„иҝӣеҲ¶жңүпјҡ

  • еҚҒиҝӣеҲ¶жңҖеёёи§Ғзҡ„пјҢеҜ№дәәжқҘиҜҙжңҖеёёз”Ёзҡ„
  • дәҢиҝӣеҲ¶и®Ўз®—жңәйҮҢйқўз”Ёзҡ„пјҢи®Ўз®—жңәзҡ„еә•еұӮиҝӣеҲ¶е°ұжҳҜдәҢиҝӣеҲ¶пјҢиҝҷжҳҜеӣ дёәи®Ўз®—жңәжҳҜз”ұйҖ»иҫ‘з”өи·Ҝз»„жҲҗзҡ„пјҢиҖҢйҖ»иҫ‘з”өи·ҜйҖҡеёёеҸӘжңүдёӨдёӘзҠ¶жҖҒпјҡејҖе…ізҡ„жҺҘйҖҡдёҺж–ӯејҖпјҢиҝҷдёӨз§ҚзҠ¶жҖҒжӯЈеҘҪеҸҜд»Ҙз”ЁдәҢиҝӣеҲ¶зҡ„ 1 е’Ң 0 иЎЁзӨәгҖӮдәҢиҝӣеҲ¶жҳҜйқһеёёеёёи§Ғзҡ„пјҢеӨ©з”ҹе°ұдёҺи®Ўз®—жңәжңүзқҖзҙ§еҜҶзҡ„е…ізі»пјҢе®ғжӯЈеҘҪиғҪд»ЈиЎЁй«ҳдҪҺз”өе№іпјҢејҖе…ізҡ„жҺҘйҖҡдёҺж–ӯејҖпјҢд№ҹдёҺйҖ»иҫ‘еӯҰдёӯзҡ„зңҹпјҢеҒҮзӣёеҗ»еҗҲпјҲеёғе°”зұ»еһӢзҡ„ True е’Ң Falseпјү
  • е…«иҝӣеҲ¶еёёи§ҒдәҺи®Ўз®—жңәпјҢз»Ҹеёёз”ЁдәҺз®ҖеҢ–дәҢиҝӣеҲ¶зҡ„и®Ўз®—пјҢдҪҶз”Ёеҫ—д№ҹзЎ®е®һжҜ”иҫғе°‘
  • еҚҒе…ӯиҝӣеҲ¶д№ҹжҳҜи®Ўз®—жңәдёӯз»ҸеёёдҪҝз”Ёзҡ„пјҢз»Ҹеёёз”ЁжқҘиЎЁзӨәеӯҳеӮЁеҷЁзҡ„ең°еқҖпјҢз”Ёеҫ—зЁҚеҫ®жҜ”е…«иҝӣеҲ¶еӨҡдёҖзӮ№гҖӮе…«иҝӣеҲ¶е’ҢеҚҒе…ӯиҝӣеҲ¶еҸӘжҳҜдёәдәҶз®ҖеҢ–дәҢиҝӣеҲ¶зҡ„еӯҳеңЁзҡ„гҖӮеңЁи®Ўз®—еҷЁеә•еұӮдёӯпјҢжҳҜеҸӘиғҪдҪҝз”ЁдәҢиҝӣеҲ¶зҡ„пјҢдҪҶжҳҜдәҢиҝӣеҲ¶еҸӘжңү 0 е’Ң 1пјҢеҠЁдёҚеҠЁе°ұиҝӣдҪҚпјҢеҜјиҮҙдәҢиҝӣеҲ¶ж•°зҡ„дҪҚж•°йҖҡеёёйғҪеҫҲеӨҡпјҢйҡҫд»Ҙи®°еҪ•пјҢжүҖд»Ҙз»ҸеёёдҪҝз”Ёе…«иҝӣеҲ¶е’ҢеҚҒе…ӯиҝӣеҲ¶жқҘз®ҖеҢ–гҖӮ
  • дёәд»Җд№ҲжҳҜеҚҒе…ӯиҝӣеҲ¶е’Ңе…«иҝӣеҲ¶жқҘеё®еҝҷпјҹиҖҢдёҚжҳҜжҲ‘д»¬жӣҙеҠ зҶҹжӮүзҡ„еҚҒиҝӣеҲ¶пјҹеҺҹеӣ еҫҲз®ҖеҚ•пјҢеҚҒе…ӯиҝӣеҲ¶зҡ„еҹәж•°жҳҜ 16пјҢе…«иҝӣеҲ¶зҡ„еҹәж•°жҳҜ 8пјҢдәҢиҝӣеҲ¶зҡ„еҹәж•°жҳҜ 2пјҢеҚҒиҝӣеҲ¶зҡ„еҹәж•°жҳҜ 10гҖӮе®ғд»¬д№ӢдёӯпјҢ16 жҳҜ 2 зҡ„ 4 ж¬Ўж–№пјҢ8 жҳҜ 2 зҡ„ 3 ж¬Ўж–№пјҲз«Ӣж–№пјүпјҢдҪҶ 10 дёҺ 2 е№¶жІЎжңүд»Җд№Ҳж•ҙж•°ж¬Ўж–№зҡ„е…ізі»гҖӮ

# еҚҒиҝӣеҲ¶дёҺд»»ж„ҸиҝӣеҲ¶д№Ӣй—ҙзҡ„дә’зӣёиҪ¬жҚў

# еҚҒиҝӣеҲ¶иҪ¬д»»ж„ҸиҝӣеҲ¶пјҡ

еҚҒиҝӣеҲ¶иҪ¬д»»ж„ҸиҝӣеҲ¶йҖҡеёёйҮҮз”ЁйҷӨеҹәж•°пјҢеҖ’еҸ–дҪҷзҡ„еҠһжі•пјҢе…·дҪ“еҒҡжі•е°ұжҳҜпјҡжҠҠиҰҒиҪ¬жҚўзҡ„ж•°йҷӨзӣ®ж ҮиҝӣеҲ¶ж•°зҡ„еҹәж•°пјҢзӣҙеҲ°ж•ҙж•°еҸҳдёә 0 дёәжӯўпјҲдёӢйқўд»Ҙ 43 иҪ¬дәҢиҝӣеҲ¶дёәдҫӢпјү

43 ж•ҙйҷӨ 2пјҢеҫ— 21пјҢдҪҷ 1

21 ж•ҙйҷӨ 2пјҢеҫ— 10пјҢдҪҷ 1

10 ж•ҙйҷӨ 2пјҢеҫ— 5пјҢдҪҷ 0

5 ж•ҙйҷӨ 2пјҢеҫ— 2пјҢдҪҷ 1

2 ж•ҙйҷӨ 2пјҢеҫ— 1пјҢдҪҷ 0

1 ж•ҙйҷӨ 2пјҢеҫ— 0пјҢдҪҷ 1

жҠҠдҪҷж•°д»ҺдёӢеҫҖдёҠиҝӣиЎҢи®°еҪ•пјҢеҫ—пјҡ101011

еҚі 43 (10) =101011 (2)

е…¶е®ғж•°пјҢе…¶е®ғиҝӣеҲ¶еҗҢзҗҶпјҢеҸӘжҳҜеҚҒе…ӯиҝӣеҲ¶иҰҒи®°еҫ—з”Ё A-F жӣҝжҚў 10-15

# д»»ж„ҸиҝӣеҲ¶иҪ¬еҚҒиҝӣеҲ¶пјҡ

д»»ж„ҸиҝӣеҲ¶иҪ¬еҚҒиҝӣеҲ¶пјҢдёҖиҲ¬з”ЁжҠҠдәҢиҝӣеҲ¶ж•°жҢүжқғеұ•ејҖгҖҒеҶҚзӣёеҠ зҡ„еҠһжі•пјҢдёҫдёӘдҫӢеӯҗпјҲ101011 иҪ¬еҚҒиҝӣеҲ¶пјү

12^5 + 02^4 + 12^3 + 02^2 + 12^1 + 12^0

=32 + 0 + 8 + 0 + 2 + 1

=43

еҚіпјҡ101011 (2) = 43 (10)

е…¶е®ғж•°пјҢе…¶е®ғиҝӣеҲ¶еҗҢзҗҶпјҢеҸӘжҳҜеҚҒе…ӯиҝӣеҲ¶иҰҒи®°еҫ—е…ҲжҠҠ A-F жӣҝжҚўжҲҗзӣёеә”зҡ„ 10-15пјҢеҶҚеҸӮдёҺиҝҗз®—

# еҚҒиҝӣеҲ¶иҪ¬д»»ж„ҸиҝӣеҲ¶пјҲе°Ҹж•°пјүпјҡ

еҚҒиҝӣеҲ¶е°Ҹж•°иҪ¬жҚўжҲҗдәҢиҝӣеҲ¶е°Ҹж•°йҮҮз”Ё вҖңд№ҳ 2 еҸ–ж•ҙпјҢйЎәеәҸжҺ’еҲ—вҖқ жі•гҖӮдёӢйқўд»ҘеҚҒиҝӣеҲ¶иҪ¬дәҢиҝӣеҲ¶пјҲе°Ҹж•°пјүдёәдҫӢпјҡ

е…·дҪ“еҒҡжі•жҳҜпјҡе…Ҳе°Ҷж•ҙж•°йғЁеҲҶдёҺе°Ҹж•°йғЁеҲҶеҲҶејҖпјҢж•ҙж•°йғЁеҲҶзӣҙжҺҘиҪ¬жҚўжҲҗдәҢиҝӣеҲ¶ж•°пјҢе°Ҹж•°йғЁеҲҶзҡ„иҪ¬жҚўеҠһжі•жҳҜпјҡз”Ё 2 д№ҳеҚҒиҝӣеҲ¶е°Ҹж•°пјҢеҸҜд»Ҙеҫ—еҲ°з§ҜпјҢе°Ҷз§Ҝзҡ„ж•ҙж•°йғЁеҲҶеҸ–еҮәпјҢеҶҚз”Ё 2 д№ҳдҪҷдёӢзҡ„е°Ҹж•°йғЁеҲҶпјҢеҸҲеҫ—еҲ°дёҖдёӘз§ҜпјҢеҶҚе°Ҷз§Ҝзҡ„ж•ҙж•°йғЁеҲҶеҸ–еҮәпјҢеҰӮжӯӨиҝӣиЎҢпјҢзӣҙеҲ°з§Ҝдёӯзҡ„е°Ҹж•°йғЁеҲҶдёәйӣ¶пјҢжҲ–иҖ…иҫҫеҲ°жүҖиҰҒжұӮзҡ„зІҫеәҰдёәжӯўгҖӮпјҲеҺҹеӣ жҳҜпјҡжңүеҫҲеӨҡе°Ҹж•°иҪ¬жҚўжҲҗдәҢиҝӣеҲ¶жҳҜж— йҷҗе°Ҹж•°пјҢеҸӘиғҪеҸ–иҲҚпјҢжүҖд»ҘдёҖиҲ¬дјҡи®ҫе®ҡдёҖдёӘзІҫеәҰпјҢиҫҫеҲ°зІҫеәҰеҗҺе°ұеҒңжӯўи®Ўз®—гҖӮпјү

дҫӢеҰӮпјҲ5.625 иҪ¬дёәдәҢиҝӣеҲ¶пјүпјҡ

еҲҶжҲҗ 5гҖҒ0.625

5 иҪ¬жҚўжҲҗдәҢиҝӣеҲ¶жҳҜпјҡ101пјҢе…Ҳж”ҫдёҖиҫ№

0.625*2 = 1.25пјҢеҸ–ж•ҙдёә 1

0.25*2 = 0.5пјҢеҸ–ж•ҙдёә 0

0.5*2 = 1пјҢеҸ–ж•ҙдёә 1

йЎәзқҖеҸ–ж•ҙпјҢеҫ— 101

жүҖд»Ҙ 0.625 (10) = 0.101 (2)

жүҖд»Ҙ 5.625 (10) = 101.625 (2)

еҰӮжһңе°Ҹж•°йғЁеҲҶжҖҺд№Ҳд№ҳйғҪдёҚдёә 0 зҡ„иҜқпјҢйӮЈе°ұжҢүд№ӢеүҚи§„е®ҡзҡ„пјҢз®—еҲ°зІҫеәҰиҰҒжұӮдёәжӯўгҖӮ

дҫӢеҰӮпјҡ0.56 иҪ¬жҚўдёәдәҢиҝӣеҲ¶пјҲдҝқз•ҷе°Ҹж•°зӮ№еҗҺ 5 дҪҚпјү

0.56*2 = 1.12пјҢеҸ–ж•ҙдёә 1

0.12*2 = 0.24пјҢеҸ–ж•ҙдёә 0

0.24*2 = 0.48пјҢеҸ–ж•ҙдёә 0

0.48*2 = 0.96пјҢеҸ–ж•ҙдёә 0

0.96*2 = 1.92пјҢеҸ–ж•ҙдёә 1
