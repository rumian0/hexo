# QtUnblockNeteaseMusic

【软件版本】v1.5.3
【测试机型】Windows
【软件大小】24.07M
【软件介绍】
一款解锁解锁某抑云中无法播放的音乐工具，基于 UnblockNeteaseMusic，软件提供了一种自建代理节点的方式让不可用的歌曲包含多个听歌源。
【使用教程】
第一部下载好压缩包并解压

第二部点开解压出来的文件夹双击点开，找到 QtUnblockNeteaseMusic.exe 文件右键以管理员身份运行

第三部运行后选择上边的高级按钮一安装证书，按照步骤安装即可

第四部将工具内的端口什么啥的全部删除再去点击应用

第五部打开网易云一设置一工具，找到 http 代理选择自定义代理，下边选择 http 代理，服务器填 127.0.0.1，端口填 8080，之后点击测试显示可用就可以了

最后关闭重新打开网易云会发现那些需要会员的歌曲前边没有会员标识了选择会员歌曲也能完整播放

注:
想要听会员歌曲必须后台运行 QtUnblockNeteaseMusic.exe 这个工具，也可以直接在高级一高级
选项一开机自启
