# API

api 如下：

# 参数说明

参数 说明 是否必须
bv 视频的 bv 号 是
qn 视频清晰度 否（默认为 80）

# 视频清晰度

qn 对应清晰度
16 360P 流畅
32 480P 清晰
64 720P 高清
80 1080P 高清

# 使用用途

可以作为不会过期的 b 站视频直链使用。

例如在网页中插入：

# 效果

<div class="videosamplex">
<video id="videoplayer" src="https://bili.api.scc.lol/?bv=BV1Tm4215773&qn=80"></video>
</div>
